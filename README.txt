Requisitos
- Visual Code Studio (con Python 3.13)
- Que el archivo este guardado con .py
- Asegurarse que dentro del archivo estén importadas las librerías “os” y deepcopy desde copy

Proyecto
El proyecto es un minijuego que consiste en llevar al ratón a su casilla de escape antes de que el gato te atrape. Dentro del tablero se encuentra un queso el cual le habilita al ratón a realizar movimientos en diagonales y también permite al ratón salir por la puerta de escape. El tablero de juego es un tablero 10x10 con algunos obstáculos.
